Aikido Journey

Proyecto final ISDI Bootcamp - 2023

Aikido Journey es un proyecto realizado para el Bootcamp de Web development full-stack de ISDI Coders en la edición de enero de 2023.

Este backend se ha realizado utilizando como servidor Node.js con Express, utilizando Mongoose como ORM para la conexión con una base de datos de MongoDB Atlas. Además, se han aplicado conceptos básicos de arquitectura DDD para la organización del proyecto.
